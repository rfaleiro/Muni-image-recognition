The Data Pipeline

An automated workflow from image recongnition to data insghts.

1. Video input

Video is captured by local webcam.

2. Process

Image recognition model captures when a new Muni arrived at local stop.

3. Store

Results are parsed and streamed into a SQLite database.

4. Data processing / Predications

Forcast is provided based trained data.

5. Visualize

Data is made available only through Flask / HTML.
